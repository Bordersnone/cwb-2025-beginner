Flexbox Navigation Bar Example

This navigation uses Flexbox to create a professional layout.

Key Flexbox Properties Used:

  • display: flex - Makes nav a flex container
  • justify-content: space-between - Pushes logo left, links right
  • align-items: center - Vertically centers items
  • gap: 30px - Adds space between nav links

Try Experimenting:

  • Change space-between to space-around or center
  • Change gap value to see different spacing
  • Add more nav links
  • Resize the window to see responsive behavior
