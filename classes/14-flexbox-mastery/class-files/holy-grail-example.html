Main Content Area

This is the famous "Holy Grail" layout - a classic web design pattern that consists of a header, footer, sidebar, and main content area. It's called "Holy Grail" because it was historically difficult to achieve with CSS, but Flexbox makes it easy!

Notice how the footer stays at the bottom of the page even when there's not much content. This is achieved by making the body a flex container with min-height: 100vh and giving the main container flex: 1.

How This Layout Works

Body (Outer Flex Container):

  • display: flex + flex-direction: column = vertical layout
  • min-height: 100vh = at least full viewport height

Main Container (Inner Flex Container):

  • display: flex = horizontal layout (default flex-direction: row)
  • flex: 1 = grows to fill remaining vertical space

Sidebar:

  • width: 200px = fixed width sidebar

Main Content:

  • flex: 1 = takes remaining horizontal space

Result: Footer always sticks to bottom, sidebar is fixed width, main content is flexible!

Try adding more content or removing content to see how the footer always stays at the bottom.
