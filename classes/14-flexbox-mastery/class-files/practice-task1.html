Task 1: Photo Gallery Challenge

Your Challenge:

Create a responsive photo gallery using Flexbox. The gallery should:

  • Display images in a flexible grid
  • Wrap to new lines when screen is narrow
  • Each image should be at least 200px wide
  • Images should grow to fill available space
  • Add a hover effect (your choice: scale, opacity, shadow, etc.)
  • Use smooth transitions

Hint: You'll need flex-wrap: wrap and flex: 1 1 200px
